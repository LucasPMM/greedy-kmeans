Para rodar o projeto sem problemas as seguintes bibliotecas devem estar instaladas:
- numpy
- sklearn

Além disso, caso problemas de permissão sejam enfrentados, basta rodar o seguinte comando:
$ chmod +x main.py

Para executar basta seguir o padrão (P = 1 é o valor default para a distância de Minkowski):
$ ./main.py "../datasets/0.data"
$ ./main.py "../datasets/1.data" 2
$ ./main.py "../datasets/2.data" 3
